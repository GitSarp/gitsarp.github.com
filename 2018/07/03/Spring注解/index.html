Spring注解

发表于 2018-07-03 | 分类于 Spring | | 阅读数

常用spring注解：

  1. @GetMapping是一个组合注解，是@RequestMapping(method = RequestMethod.GET)的缩写。@PostMapping同理。

  2. @Resource和@Autowired的区别:

    • @Resource默认按照名称方式进行bean匹配，@Autowired默认按照类型方式进行bean匹配
    • @Resource(import javax.annotation.Resource;)是J2EE的注解，@Autowired( import org.springframework.beans.factory.annotation.Autowired;)是Spring的注解

  3. @Valid (在Controller响应方法的form对应实体入参上加入@Valid注解，在form对应实体的字段上添加@NotNull等注解)

  1. lombok注解：可以不用生成get、set和构造方法，idea中安装lombok-plugin插件，另外勾选enable annotation processing
